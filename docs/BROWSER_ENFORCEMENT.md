# Browser upload enforcement

## Managed browsers

Company DLP v11 contains two browser packages:

- `browser-extension`: Chrome, Edge, Brave, Opera, Vivaldi, and other Chromium-based browsers that accept Chromium extensions.
- `firefox-extension`: Firefox.

The protection blocks user file selection, `HTMLInputElement.showPicker()`, File System Access pickers, file input changes, file/form submission, file drag-and-drop anywhere in the page, and file/image paste. It also blocks network calls when the payload still contains an actual JavaScript `File` object or a `FormData` entry containing a `File`.

The guard intentionally does not classify every `ArrayBuffer`, WebSocket packet, WebRTC packet, `Blob`, or worker message as a file. Modern web applications use binary traffic for ordinary operation. Blocking all binary traffic caused periodic false upload alerts and can break WhatsApp, Telegram, cloud tools, and AI sites even when the user did not upload a file.

## Firefox development loading

1. Open `about:debugging#/runtime/this-firefox`.
2. Select **Load Temporary Add-on**.
3. Choose `firefox-extension/manifest.json`.
4. Reload any page that was already open.

Temporary Firefox add-ons are removed when Firefox exits. Production must use Firefox enterprise `ExtensionSettings` to force-install and lock the signed extension.

## Arbitrary or unmanaged browsers

A browser extension cannot protect a browser where the extension is absent, disabled, or unsupported. Production must allow only managed browsers and block all other browser executables using IT controls such as Windows Defender Application Control (WDAC) or AppLocker. Employees must use Standard User accounts and must not have permission to install or run unapproved browsers.

## Expected alert behavior

- No alert is shown merely because a page contains an upload control.
- No repeating alert is generated by ordinary background network traffic.
- One alert is shown for a real user attempt such as file picker, file drop, or file paste.
- Related signals from the same attempt are grouped and deduplicated for at least five seconds.
